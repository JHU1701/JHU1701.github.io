Realistic Lanterns Mod

The Realistic Lanterns Mod is a mod that adds dynamic lighting to minecraft and incorporates elements of raytracing and basic lighting principles to add an element of realism to the game. The mod is written in Java and utilizes different minecraft libraries that allow the developer to access the base elements of the game. While creating this mod, I needed to learn about the principles of raytracing and I needed to understand how to incoporate the different libraries into my mod. The lantern itself functions using an internal state machine with variables that can be changed by accessing the config folder. The project can be found here.
